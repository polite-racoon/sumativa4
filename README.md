Sigue los siguientes pasos para descargar y ejecutar el proyecto:

En Windows:

```
git clone https://github.com/polite-racoon/sumativa4.git
cd sumativa4
gradle.bat run
```

En Linux y macOS:

```
git clone https://github.com/polite-racoon/sumativa4.git
cd sumativa4
chmod +x gradlew
./gradlew run
```
